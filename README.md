# eurostat-tourism-dataviz
Project Title: Tourism and Culture in Europe (2015–2024)

Data Source: Eurostat — Nights spent at tourist accommodation establishments (tour_occ_ninat)

Scope: Germany, France, Italy, Spain, Greece, Netherlands, Austria

Reproducibility Steps

Download the dataset from Eurostat (tour_occ_ninat) as .xlsx.

Apply filters: selected countries + years 2015–2024.

Clean the file in Apple Numbers (macOS):

remove metadata rows

remove empty columns created during export

keep numeric values only (remove estimated “e” markers)

Create visualizations:

Datawrapper: line chart (2015–2024)

Flourish: slope chart (2019 vs 2024)

Tableau Public: map (2024)

Tools Used

Apple Numbers (data cleaning)

Datawrapper (line chart)

Flourish (slope chart)

Tableau Public (map)

ChatGPT + Google Gemini (writing support)
